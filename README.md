marc_scripts
============

marc related scripts especial for cmarc and some convert

marc_to_utf8.pl
===============

The basic usage:

    marc_scripts$ perl marc_to_utf8.pl
    Usage: marc_to_utf8.pl [options]
    Options:
        -i  FILE    input file.
        -o  FILE    output file.
        -f  ENCODE  Convert characters from encoding.
        -d          Debug mode.
        -h          Print this summary.
